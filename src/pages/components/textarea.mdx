## How to use it

### Default

By default, textarea height is fixed.

import Default from '../../examples/forms/textarea/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/forms/textarea/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Auto-expand

Textarea height can be configured to resize automatically based on user input.

import AutoExpand from '../../examples/forms/textarea/AutoExpand.tsx';
import AutoExpandCode from '!!raw-loader!../../examples/forms/textarea/AutoExpand.tsx';

<CodeExample code={AutoExpandCode}>
  <AutoExpand />
</CodeExample>

### Hidden label

Textarea labels can be hidden.

import Hidden from '../../examples/forms/textarea/Hidden.tsx';
import HiddenCode from '!!raw-loader!../../examples/forms/textarea/Hidden.tsx';

<CodeExample code={HiddenCode}>
  <Hidden />
</CodeExample>

### Manual resizing

Drag handles can be applied to a textarea for manual resizing.

import DragHandles from '../../examples/forms/textarea/DragHandles.tsx';
import DragHandlesCode from '!!raw-loader!../../examples/forms/textarea/DragHandles.tsx';

<CodeExample code={DragHandlesCode}>
  <DragHandles />
</CodeExample>

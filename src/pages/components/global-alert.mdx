## How to use it

### Default

The Global alert is the first element of a page and pushes content down. It is read first
by a screen reader and is persistent until the user dismisses it or the issue has been resolved.

import Default from '../../examples/notifications/global-alert/GlobalAlertDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/notifications/global-alert/GlobalAlertDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Type

Types indicate what kind of message is in the Global alert and help the user understand
how to respond. They can be info, warning, error, or success.

import Types from '../../examples/notifications/global-alert/GlobalAlertTypes.tsx';

## How to use it well

### Clearly actionable

Provide one button to address the issue, and an optional way to “learn more”.

<BestPractice>
  <Do
    imageSource={props.data.globalAlertClearlyActionable.childFile.childImageSharp.gatsbyImageData}
  >
    <p>Global alerts should have one clear action</p>
  </Do>
</BestPractice>

### Allow users to dismiss

Allow users to close the Global alert if it does not impede the use of the product.

<BestPractice>
  <Do imageSource={props.data.globalAlertDismissable.childFile.childImageSharp.gatsbyImageData}>
    <p>Passive, informational messages should be dismissible</p>
  </Do>
</BestPractice>

### Do not time out

Global alerts are persistent. They should either be dismissed temporarily
by the user or dismissed automatically when the issue is resolved.

<BestPractice>
  <Do imageSource={props.data.globalAlertPersistent.childFile.childImageSharp.gatsbyImageData}>
    <p>
      Global alerts are always visible until the issue has been resolved or the alert is manually
      dismissed by the user
    </p>
  </Do>
</BestPractice>

## API

The Global alert component follows this structure:

<!-- prettier-ignore -->
```jsx
<GlobalAlert>
  <GlobalAlert.Content>
    {/* main text content and nested anchors */}

    <GlobalAlert.Title>
      {/* a title for global alert */}
    </GlobalAlert.Title>

  </GlobalAlert.Content>

  <GlobalAlert.Button>
    {/* optional action for global alert */}
  </GlobalAlert.Button>

  <GlobalAlert.Close aria-label="Close Button" />
</GlobalAlert>
```

## How to use

### Type

Icon Buttons have 3 types: basic, outline, and primary. The default type is basic.

import IconButtonTypes from '../../examples/button/IconButtonTypes.tsx';
import IconButtonTypesCode from '!!raw-loader!../../examples/button/IconButtonTypes.tsx';

<CodeExample code={IconButtonTypesCode}>
  <IconButtonTypes />
</CodeExample>

### Size

Icon Buttons can be small, medium, or large. The default size is medium.

import IconButtonSizes from '../../examples/button/IconButtonSizes.tsx';
import IconButtonSizesCode from '!!raw-loader!../../examples/button/IconButtonSizes.tsx';

<CodeExample code={IconButtonSizesCode}>
  <IconButtonSizes />
</CodeExample>

### Shape

Icon Buttons are circular by default, but they can be squares too.

import IconButtonShapes from '../../examples/button/IconButtonShapes.tsx';
import IconButtonShapesCode from '!!raw-loader!../../examples/button/IconButtonShapes.tsx';

<CodeExample code={IconButtonShapesCode}>
  <IconButtonShapes />
</CodeExample>

### Danger

Danger Icon Buttons indicate destructive actions.

import IconButtonDanger from '../../examples/button/IconButtonDanger.tsx';
import IconButtonDangerCode from '!!raw-loader!../../examples/button/IconButtonDanger.tsx';

### Disabled

Disabled buttons prevent user interaction.

## How to use it well

### Don't leave users in the dark

  <Do imageSource={props.data.iconButtonTooltipDo.children[0].publicURL}>

  </Do>

## How to use it

### Default

The typical usage of an Icon button component.

import IconButtonDefault from '../../examples/button/IconButtonDefault.tsx';
import IconButtonDefaultCode from '!!raw-loader!../../examples/button/IconButtonDefault.tsx';

<CodeExample code={IconButtonDefaultCode}>
  <IconButtonDefault />
</CodeExample>

### Danger

Use danger styling for Icon buttons that enable destructive action.

import IconButtonDanger from '../../examples/button/IconButtonDanger.tsx';
import IconButtonDangerCode from '!!raw-loader!../../examples/button/IconButtonDanger.tsx';

<CodeExample code={IconButtonDangerCode}>
  <IconButtonDanger />
</CodeExample>

### Disabled

A disabled Icon button prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may not
be read aloud by a screenreader.

import IconButtonDisabled from '../../examples/button/IconButtonDisabled.tsx';
import IconButtonDisabledCode from '!!raw-loader!../../examples/button/IconButtonDisabled.tsx';

<CodeExample code={IconButtonDisabledCode}>
  <IconButtonDisabled />
</CodeExample>

### Shape

Icon buttons are circular by default, but they can be squares too.

import IconButtonShapes from '../../examples/button/IconButtonShapes.tsx';
import IconButtonShapesCode from '!!raw-loader!../../examples/button/IconButtonShapes.tsx';

<CodeExample code={IconButtonShapesCode}>
  <IconButtonShapes />
</CodeExample>

### Size

Icon buttons can be small, medium, or large. The default size is medium.

import IconButtonSizes from '../../examples/button/IconButtonSizes.tsx';
import IconButtonSizesCode from '!!raw-loader!../../examples/button/IconButtonSizes.tsx';

<CodeExample code={IconButtonSizesCode}>
  <IconButtonSizes />
</CodeExample>

### Type

Icon buttons have 3 types: basic, outline, and primary. The default type is basic.

import IconButtonTypes from '../../examples/button/IconButtonTypes.tsx';
import IconButtonTypesCode from '!!raw-loader!../../examples/button/IconButtonTypes.tsx';

## How to use it

### Default

Use Paragraph to style and format `<p>` elements.

import Default from '../../examples/typography/paragraph/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/typography/paragraph/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Size

Paragraphs come in small, medium, and large. The default size is medium. Size
only controls space between paragraphs. Select a
[typography](/components/typography) scale which is sized to match.

import Size from '../../examples/typography/paragraph/Size.tsx';
import SizeCode from '!!raw-loader!../../examples/typography/paragraph/Size.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

### Avatar

Medium and large Tags can contain an Avatar.

### Color

Tags can be colored using a [theme
palette](https://zendeskgarden.github.io/react-components/theming/#palette) hue
or hex value.

### Shape

Tags are rectangular by default, but they can be pill or round too. Round tags
are reserved for cases of 2 digits or less.

### Size

Tags come in small, medium, and large. The default size is medium.

### Mixed weight

Tag content is regular weight by default, but can be bold too.

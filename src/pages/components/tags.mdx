## API

The Tag component follows this structure:

```jsx
<Tag>
  <Tag.Avatar /> {/* optional */}
  {/* tag text */}
  <Tag.Close /> {/* optional */}
</Tag>
```

### Tag

<Component components={props.data.mdx.components} componentName="tag" />

The Tag component applies appropriate styles to itself and its descendents based on the props and
child elements provided.

<PropSheet components={props.data.mdx.components} componentName="tag" />

#### Tag.Avatar

<Component components={props.data.mdx.subcomponents} componentName="tag.avatar" />

The Avatar component displays the child media with appropriate styles applied. Nest it within the [Tag](#tag)
component as the first child.

<PropSheet components={props.data.mdx.subcomponents} componentName="tag.avatar" />

#### Tag.Close

<Component components={props.data.mdx.subcomponents} componentName="tag.close" />

The Close component displays an icon with appropriate styles and accessibility attributes to allow tag
dismissal. Nest it within the [Tag](#tag) component as the last child.

<PropSheet components={props.data.mdx.subcomponents} componentName="tag.close" />

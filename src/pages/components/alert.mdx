## How to use it

### Default

By default, an Alert will be read first by a screen reader and is persistent
until the user dismisses it.

import Default from '../../examples/notifications/alert/AlertDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/notifications/alert/AlertDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Type

Alert types help communicate the tone of the message. They can be Success,
Warning, Danger, or Info.

import TypeCode from '!!raw-loader!../../examples/notifications/alert/AlertType.tsx';

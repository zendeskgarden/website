### Size

### Hidden Label

The label can be hidden.

import ComboboxHidden from '../../examples/dropdowns/combobox/Hidden.tsx';
import ComboboxHiddenCode from '!!raw-loader!../../examples/dropdowns/combobox/Hidden.tsx';

<CodeExample code={ComboboxHiddenCode}>
  <ComboboxHidden />
</CodeExample>

### Selection Logic

The combobox can be customized to include selection logic that highlights the first relevant
result as the user types, allowing the user to press Enter to select the item.
If this logic is not included, the user must manually select a result from the list.

## API

The Combobox component follows this structure:

```jsx
<Dropdown>
  <Field>
    <Label />
    <Hint />
    <Combobox />
  </Field>
  <Menu>
    <Item />
    <Item />
    {/* etc. */}
  </Menu>
</Dropdown>
```

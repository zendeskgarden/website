## How to use it

### Default

As the user types, options appear in the list box that match what the user is typing.

import ComboboxDefault from '../../examples/dropdowns/combobox/Default.tsx';
import ComboboxDefaultCode from '!!raw-loader!../../examples/dropdowns/combobox/Default.tsx';

<CodeExample code={ComboboxDefaultCode}>
  <ComboboxDefault />
</CodeExample>

### Hidden Label

The label can be hidden as needed.

import ComboboxHidden from '../../examples/dropdowns/combobox/Hidden.tsx';
import ComboboxHiddenCode from '!!raw-loader!../../examples/dropdowns/combobox/Hidden.tsx';

<CodeExample code={ComboboxHiddenCode}>
  <ComboboxHidden />
</CodeExample>

### Media

You can place the magnifying glass icon at the start of the input when using the Combobox for search.

import ComboboxMedia from '../../examples/dropdowns/combobox/Media.tsx';
import ComboboxMediaCode from '!!raw-loader!../../examples/dropdowns/combobox/Media.tsx';

<CodeExample code={ComboboxMediaCode}>
  <ComboboxMedia />
</CodeExample>

### Selection Logic

The combobox can can be customized to include selection logic that highlights the first relevant
result as the user types, allowing the user to hit enter to fully select the item. If this logic
is not included, the user must manually select a result from the list.

import ComboboxSelection from '../../examples/dropdowns/combobox/Selection.tsx';
import ComboboxSelectionCode from '!!raw-loader!../../examples/dropdowns/combobox/Selection.tsx';

<CodeExample code={ComboboxSelectionCode}>
  <ComboboxSelection />
</CodeExample>

### Size

Combobox can be displayed in either default or compact size.

import ComboboxSize from '../../examples/dropdowns/combobox/Size.tsx';
import ComboboxSizeCode from '!!raw-loader!../../examples/dropdowns/combobox/Size.tsx';

<CodeExample code={ComboboxSizeCode}>

## API

The Combobox component follows this structure:

```jsx
<Dropdown>
  <Field>
    <Label />
    <Hint />
    <Combobox />
  </Field>
  <Menu>
    <Item />
    <Item />
    {/* etc. */}
  </Menu>
</Dropdown>
```

## How to use it

### Default

The Color picker allows users to choose colors using sliders and input fields.

import ColorpickerDefault from '../../examples/colorpickers/ColorpickerDefault.tsx';
import ColorpickerDefaultCode from '!!raw-loader!../../examples/colorpickers/ColorpickerDefault.tsx';

<CodeExample code={ColorpickerDefaultCode}>
  <ColorpickerDefault />
</CodeExample>

### Dialog

The Color picker dialog allows users to use a color picker within a dialog.

import ColorpickerDialogDefault from '../../examples/colorpickers/ColorpickerDialogDefault.tsx';
import ColorpickerDialogDefaultCode from '!!raw-loader!../../examples/colorpickers/ColorpickerDialogDefault.tsx';

<CodeExample code={ColorpickerDialogDefaultCode}>
  <ColorpickerDialogDefault />
</CodeExample>

### Form input

The Color picker dialog can be used in an input group.

import ColorpickerDialogFormInput from '../../examples/colorpickers/ColorpickerDialogFormInput.tsx';
import ColorpickerDialogFormInputCode from '!!raw-loader!../../examples/colorpickers/ColorpickerDialogFormInput.tsx';

  <ColorpickerDialogFormInput />

The Color picker can be displayed using a Trigger, e.g. an icon button.

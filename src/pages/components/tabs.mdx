## How to use it

### Default

The basic usage of a Tab component.

import TabsDefault from '../../examples/tabs/TabsDefault.tsx';
import TabsDefaultCode from '!!raw-loader!../../examples/tabs/TabsDefault.tsx';

<CodeExample code={TabsDefaultCode}>
  <TabsDefault />
</CodeExample>

### Disabled

Individual tabs can be disabled to prevent users from interacting with them.

import TabsDisabled from '../../examples/tabs/TabsDisabled.tsx';
import TabsDisabledCode from '!!raw-loader!../../examples/tabs/TabsDisabled.tsx';

<CodeExample code={TabsDisabledCode}>
  <TabsDisabled />
</CodeExample>

### Orientation

Tab layout is horizontal by default and can also stack vertically.

import TabsVertical from '../../examples/tabs/TabsVertical.tsx';
import TabsVerticalCode from '!!raw-loader!../../examples/tabs/TabsVertical.tsx';

<CodeExample code={TabsVerticalCode}>
  <TabsVertical />
</CodeExample>

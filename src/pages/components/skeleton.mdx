## How to use it

### Default

Skeleton loaders are the same color as the text color at 10% opacity.

import Default from '../../examples/loaders/skeleton/SkeletonDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/loaders/skeleton/SkeletonDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Light

Skeleton loaders can be inverted for use on dark backgrounds.

import Light from '../../examples/loaders/skeleton/SkeletonLight.tsx';
import LightCode from '!!raw-loader!../../examples/loaders/skeleton/SkeletonLight.tsx';

<CodeExample code={LightCode}>
  <Light />
</CodeExample>

### Size

By default, a Skeleton loader's height is 60% of the font size. To
make placeholders for other components, such as buttons, height and width can be
explicitly set.

import Size from '../../examples/loaders/skeleton/SkeletonSize.tsx';
import SizeCode from '!!raw-loader!../../examples/loaders/skeleton/SkeletonSize.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

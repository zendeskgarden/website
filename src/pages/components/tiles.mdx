## How to use it

### Default

The typical usage of a Tiles component.

import Default from '../../examples/forms/tiles/TilesDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/forms/tiles/TilesDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Description

A description adds additional context to the Tile label.

import Description from '../../examples/forms/tiles/TilesDescription.tsx';
import DescriptionCode from '!!raw-loader!../../examples/forms/tiles/TilesDescription.tsx';

<CodeExample code={DescriptionCode}>
  <Description />
</CodeExample>

### Disabled

Disabled Tiles are not interactive and may not be perceived by a screen reader.

import Disabled from '../../examples/forms/tiles/TilesDisabled.tsx';
import DisabledCode from '!!raw-loader!../../examples/forms/tiles/TilesDisabled.tsx';

<CodeExample code={DisabledCode}>
  <Disabled />
</CodeExample>

### Layout

Content within a single Tile is vertically centered by default. The layout can
be adjusted by setting the property `isCentered` to false.

import Layout from '../../examples/forms/tiles/TilesLayout.tsx';
import LayoutCode from '!!raw-loader!../../examples/forms/tiles/TilesLayout.tsx';

<CodeExample code={LayoutCode}>
  <Layout />
</CodeExample>

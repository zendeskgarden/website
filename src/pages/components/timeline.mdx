### Default

The timeline component is made up of individual timeline items. Item content appears to the right of the item marker by default.

## How to use it well

### Nest content when necessary

Components such as links, buttons, and images can be nested in each timeline item.

<BestPractice>
  <Do>
    <p>Do include images and other nested content if appropriate.</p>
  </Do>
</BestPractice>

### Make interactive elements explicit

Creating a visual affordance for a link or action is preferred to making the entire item interactive.

<BestPractice>
  <Do>
    <p>Create explicit visual affordances for interactive items.</p>
  </Do>
  <Dont>
    <p>Don’t rely on hover states to show that items are interactive.</p>
  </Dont>
</BestPractice>

### Use correct hierarchy

Item content can be replaced with any text style. Make sure use appropriate hierarchy when designing your content.

<BestPractice>
  <Do>
    <p>Use balanced and appropriate typographic hierarchy.</p>
  </Do>
  <Dont>
    <p>Don’t mix up typographic hierarchy when customizing item content.</p>
  </Dont>
</BestPractice>

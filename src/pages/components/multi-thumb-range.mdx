## How to use it

### Default

A Multi-thumb range has two thumbs.

import Default from '../../examples/forms/multithumbrange/MultiThumbRangeDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/forms/multithumbrange/MultiThumbRangeDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Disabled

A disabled Multi-thumb range prevents user interaction. It doesn’t appear in the tab order,
can’t receive focus, and may not be read aloud by a screenreader.

import Disabled from '../../examples/forms/multithumbrange/MultiThumbRangeDisabled.tsx';
import DisabledCode from '!!raw-loader!../../examples/forms/multithumbrange/MultiThumbRangeDisabled.tsx';

<CodeExample code={DisabledCode}>
  <Disabled />
</CodeExample>

### Hidden label

Multi-thumb range labels can be hidden.

import Hidden from '../../examples/forms/multithumbrange/MultiThumbRangeHidden.tsx';
import HiddenCode from '!!raw-loader!../../examples/forms/multithumbrange/MultiThumbRangeHidden.tsx';

<CodeExample code={HiddenCode}>
  <Hidden />
</CodeExample>

### Hint text

Add context to the Multi-thumb range label using Hint.

import HintCode from '!!raw-loader!../../examples/forms/multithumbrange/MultiThumbRangeHint.tsx';

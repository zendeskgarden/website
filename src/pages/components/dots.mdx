## How to use it

### Default

The default application of a Dots loader includes accessibility attributes that
identify it as an indeterminate progress bar.

import Default from '../../examples/loaders/dots/DotsDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/loaders/dots/DotsDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Color

By default, color is inherited from the parent element unless specified.

import Color from '../../examples/loaders/dots/DotsColor.tsx';
import ColorCode from '!!raw-loader!../../examples/loaders/dots/DotsColor.tsx';

<CodeExample code={ColorCode}>
  <Color />
</CodeExample>

### Size

By default, size is inherited from font size unless specified.

import Size from '../../examples/loaders/dots/DotsSize.tsx';
import SizeCode from '!!raw-loader!../../examples/loaders/dots/DotsSize.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

### Regular weight label

Text input labels are emphasized by default. They can be regular weight using the `isRegular`
property.

Regular labels are not recommended for text inputs, as they flatten visual hierarchy making
scanning more difficult.

import RegularWeightLabelCode from '!!raw-loader!../../examples/inputs/RegularWeightLabel.tsx';

## API

The Menu component follows this structure:

```jsx
<Menu>
  <Item />
  <Item />
  {/* etc. */}
</Menu>
```

### Menu

<Component components={props.data.mdx.components} componentName="menu" />

A Menu manages placement, positioning and layout of its child Item components.

<PropSheet components={props.data.mdx.components} componentName="menu" />

### Item

<Component components={props.data.mdx.components} componentName="item" />

Nest an Item within a Menu component. It requires a value prop.

<PropSheet components={props.data.mdx.components} componentName="item" />

#### Item.Meta

<Component components={props.data.mdx.subcomponents} componentName="item.meta" />

An ItemMeta provides additional information about a menu item. Nest it within an Item component.

<PropSheet components={props.data.mdx.subcomponents} componentName="item.meta" />

### ItemGroup

<Component components={props.data.mdx.components} componentName="itemgroup" />

Provides a semantic grouping to its child Item components. Optionally configures its Items as
selectable.

<PropSheet components={props.data.mdx.components} componentName="itemgroup" />

### Separator

<Component components={props.data.mdx.components} componentName="separator" />

A semantic separator between Item components.

<PropSheet components={props.data.mdx.components} componentName="separator" />

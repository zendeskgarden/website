## How to use it

### Default

Works in combination with a Trigger to display a list of menu items.

import MenuDefault from '../../examples/dropdowns/menu/Default.tsx';
import MenuDefaultCode from '!!raw-loader!../../examples/dropdowns/menu/Default.tsx';

<CodeExample code={MenuDefaultCode}>
  <MenuDefault />
</CodeExample>

### Arrow

An arrow can be added to the Menu to give the user a sense of what element is triggering it.

import MenuArrow from '../../examples/dropdowns/menu/Arrow.tsx';
import MenuArrowCode from '!!raw-loader!../../examples/dropdowns/menu/Arrow.tsx';

<CodeExample code={MenuArrowCode}>
  <MenuArrow />
</CodeExample>

### Disabled

Items within a Menu can be disabled to limit a user from interacting with them.

import MenuDisabled from '../../examples/dropdowns/menu/Disabled.tsx';
import MenuDisabledCode from '!!raw-loader!../../examples/dropdowns/menu/Disabled.tsx';

<CodeExample code={MenuDisabledCode}>
  <MenuDisabled />
</CodeExample>

### Media items

Media elements add even more context to a Menu.

import MenuMedia from '../../examples/dropdowns/menu/Media.tsx';
import MenuMediaCode from '!!raw-loader!../../examples/dropdowns/menu/Media.tsx';

<CodeExample code={MenuMediaCode}>
  <MenuMedia />
</CodeExample>

### Meta

Meta text can be paired with Menu items for additional clarity.

import MenuMeta from '../../examples/dropdowns/menu/Meta.tsx';
import MenuMetaCode from '!!raw-loader!../../examples/dropdowns/menu/Meta.tsx';

<CodeExample code={MenuMetaCode}>
  <MenuMeta />
</CodeExample>

### Nested

Menus can contain nested levels for additional categorization of menu items.

import MenuNested from '../../examples/dropdowns/menu/Nested.tsx';
import MenuNestedCode from '!!raw-loader!../../examples/dropdowns/menu/Nested.tsx';

<CodeExample code={MenuNestedCode}>
  <MenuNested />
</CodeExample>

### Placement

Menu placement can be oriented around a trigger element in different positions. The default
placement is `bottom-start`. This example demonstrates the `top-start` placement.

import MenuPlacement from '../../examples/dropdowns/menu/Placement.tsx';
import MenuPlacementCode from '!!raw-loader!../../examples/dropdowns/menu/Placement.tsx';

<CodeExample code={MenuPlacementCode}>
  <MenuPlacement />
</CodeExample>

### Size

Can be default or compact in size. This reduces vertical padding on menu items.

import MenuSize from '../../examples/dropdowns/menu/Size.tsx';
import MenuSizeCode from '!!raw-loader!../../examples/dropdowns/menu/Size.tsx';

<CodeExample code={MenuSizeCode}>
  <MenuSize />
</CodeExample>

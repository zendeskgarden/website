## How to use it

### Default

Use Code to style and format `<code>` elements.

import Default from '../../examples/typography/code/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/typography/code/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Color

You can also change background and text color.

import Color from '../../examples/typography/code/Color.tsx';
import ColorCode from '!!raw-loader!../../examples/typography/code/Color.tsx';

<CodeExample code={ColorCode}>
  <Color />
</CodeExample>

### Size

Code comes in small, medium, and large. By default, size is inherited from
the parent element.

import Size from '../../examples/typography/code/Size.tsx';
import SizeCode from '!!raw-loader!../../examples/typography/code/Size.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

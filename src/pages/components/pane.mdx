that you set.

## Customization

### Vertical collapse

Panes can be collapsed or expanded by using the splitter together with an
[Icon button](/components/icon-button). Vertical orientation uses a small circular
icon button positioned on top of the splitter by default in both collapsed and
expanded states. [Tooltips](/components/tooltip) should be used to describe the action.

import PaneVerticalCollapse from '../../examples/pane/PaneVerticalCollapse.tsx';
import PaneVerticalCollapseCode from '!!raw-loader!../../examples/pane/PaneVerticalCollapse.tsx';

<CodeExample code={PaneVerticalCollapseCode}>
  <PaneVerticalCollapse />
</CodeExample>

### Horizontal collapse

Horizontal orientation also uses a small [Icon button](/components/icon-button) to
collapse the pane. When the pane is collapsed, a stretched [Button](/components/button)
is used to indicate an expandable state of the pane.

import PaneHorizontalCollapse from '../../examples/pane/PaneHorizontalCollapse.tsx';
import PaneHorizontalCollapseCode from '!!raw-loader!../../examples/pane/PaneHorizontalCollapse.tsx';

<CodeExample code={PaneHorizontalCollapseCode}>
  <PaneHorizontalCollapse />
</CodeExample>

### Icon button position

Icon button can be positioned at the start, center or on the end of the splitter.

import IconButtonPosition from '../../examples/pane/IconButtonPosition.tsx';
import IconButtonPositionCode from '!!raw-loader!../../examples/pane/IconButtonPosition.tsx';

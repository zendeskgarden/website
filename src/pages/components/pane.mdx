## How to use it

### Vertical

Panes can be used in a vertical orientation. The panes resize up to the values
that you set.

import PaneVertical from '../../examples/pane/PaneVertical.tsx';
import PaneVerticalCode from '!!raw-loader!../../examples/pane/PaneVertical.tsx';

<CodeExample code={PaneVerticalCode}>
  <PaneVertical />
</CodeExample>

### Horizontal

Panes can be used in a horizontal orientation as well.

import PaneHorizontal from '../../examples/pane/PaneHorizontal.tsx';
import PaneHorizontalCode from '!!raw-loader!../../examples/pane/PaneHorizontal.tsx';

<CodeExample code={PaneHorizontalCode}>
  <PaneHorizontal />
</CodeExample>

### Grid

Panes can be resized in both vertical and horizontal directions for a complex
grid flexible layout.

import PaneGrid from '../../examples/pane/PaneGrid.tsx';

## Customization

Panes can be collapsed or expanded by using the [SplitterButton](#panesplitterbutton).

### Vertical collapse

import PaneVerticalCollapseCode from '!!raw-loader!../../examples/pane/PaneVerticalCollapse.tsx';

  <PaneVerticalCollapse />
</CodeExample>

### Horizontal collapse

import PaneHorizontalCollapse from '../../examples/pane/PaneHorizontalCollapse.tsx';
import PaneHorizontalCollapseCode from '!!raw-loader!../../examples/pane/PaneHorizontalCollapse.tsx';

<CodeExample code={PaneHorizontalCollapseCode}>
  <PaneHorizontalCollapse />

### Splitter button position

The [SplitterButton](#panesplitterbutton) can be positioned at the start, center or end of the
splitter. Its default position is at the start when used in vertical orientation and at the
center when used in horizontal orientation.

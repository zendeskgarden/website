---
title: Toggle icon button
description: >
  A Toggle Icon Button (like a Toggle Button) lets users turn something on
  and off like a light switch. It is used for repeated or persistent actions
  on a page and lacks a visible label to simplify the UI.
package: '@zendeskgarden/react-buttons'
components:
  - buttons/src/elements/ToggleIconButton.tsx
---

<Usage>
  <Use>
    <ul>
      <li>To see or compare the results of a settings change</li>
      <li>To cause an immediate effect, such as making text bold</li>
      <li>To simplify the appearance of repeated or persistent actions on a page</li>
      <li>To enable action in a toolbar</li>
    </ul>
  </Use>
  <Misuse>
    <ul>

        </Markdown>
      </li>
      <li>
        <Markdown>
          To include an icon and a label, use a [Button](/components/button#media) with a start or
          end icon
        </Markdown>
      </li>
    </ul>
  </Misuse>
</Usage>

### Disabled

A disabled Toggle Icon Button prevents user interaction. It doesn’t appear in
the tab order, can’t receive focus, and may not be read aloud by a
screenreader.

Toggle Icon Buttons are circular by default, but they can be squares too.

Toggle Icon Buttons can be small, medium, or large. The default size is medium.

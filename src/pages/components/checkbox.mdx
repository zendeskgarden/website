<Usage>
  <Use>
    <ul>
      <li>To let users compare options from a list and select all, any, or none of those items</li>
      <li>To turn a single option on or off</li>
    </ul>
  </Use>
  <Misuse>
    <ul>
      <li>
        <Markdown>
          To give the user a mutually exclusive choice, use the [Radio](/components/radio) component
          instead
        </Markdown>
      </li>
      <li>
        <Markdown>
          To let users activate an option that takes effect immediately, use a
          [Toggle](/components/toggle) instead
        </Markdown>
      </li>
    </ul>
  </Misuse>
</Usage>

## How to use it

### Default

A Checkbox’s label is part of its touch target.

import Default from '../../examples/forms/checkbox/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/forms/checkbox/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Disabled

A disabled Checkbox prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may not
read aloud by a screenreader.

import Disabled from '../../examples/forms/checkbox/Disabled.tsx';
import DisabledCode from '!!raw-loader!../../examples/forms/checkbox/Disabled.tsx';

<CodeExample code={DisabledCode}>
  <Disabled />
</CodeExample>

### Hidden

Checkbox labels can be hidden.

import Hidden from '../../examples/forms/checkbox/Hidden.tsx';
import HiddenCode from '!!raw-loader!../../examples/forms/checkbox/Hidden.tsx';

<CodeExample code={HiddenCode}>
  <Hidden />
</CodeExample>

### Hint text

Use Hint to add more context to the label.

import Hint from '../../examples/forms/checkbox/Hint.tsx';
import HintCode from '!!raw-loader!../../examples/forms/checkbox/Hint.tsx';

<CodeExample code={HintCode}>
  <Hint />
</CodeExample>

### Indeterminate

If the children of a Checkbox have different states (some on, some off) the parent Checkbox will
show an indeterminate state.

import Indeterminate from '../../examples/forms/checkbox/Indeterminate.tsx';
import IndeterminateCode from '!!raw-loader!../../examples/forms/checkbox/Indeterminate.tsx';

<CodeExample code={IndeterminateCode}>
  <Indeterminate />
</CodeExample>

### Regular weight label

Checkbox labels are bold by default, but they can be regular weight too.

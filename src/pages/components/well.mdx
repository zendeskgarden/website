## How to use it

### Default

The typical usage of a Well component.

import Default from '../../examples/notifications/well/WellDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/notifications/well/WellDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Floating

A floating Well has a drop shadow that lifts the content off the page.

import Floating from '../../examples/notifications/well/WellFloating.tsx';
import FloatingCode from '!!raw-loader!../../examples/notifications/well/WellFloating.tsx';

<CodeExample code={FloatingCode}>
  <Floating />
</CodeExample>

### Multi-line

A multi-line Well allows for a title and long-form content.

import Multiline from '../../examples/notifications/well/WellMultiline.tsx';
import MultilineCode from '!!raw-loader!../../examples/notifications/well/WellMultiline.tsx';

<CodeExample code={MultilineCode}>
  <Multiline />
</CodeExample>

### Recessed

A recessed Well has a background to differentiate it from the rest of the page.

import Recessed from '../../examples/notifications/well/WellRecessed.tsx';
import RecessedCode from '!!raw-loader!../../examples/notifications/well/WellRecessed.tsx';

<CodeExample code={RecessedCode}>
  <Recessed />
</CodeExample>

## How to use it

### Default

Use Kbd to style and format `<kbd>` elements.

import Default from '../../examples/typography/kbd/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/typography/kbd/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Size

Kbd comes in small, medium, and large. By default, size is inherited from
the parent element.

import Size from '../../examples/typography/kbd/Size.tsx';
import SizeCode from '!!raw-loader!../../examples/typography/kbd/Size.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

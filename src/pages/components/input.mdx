## How to use it

### Default

An Input comes with a label indicating what goes into the field.

import Input from '../../examples/forms/inputs/Input.tsx';
import InputCode from '!!raw-loader!../../examples/forms/inputs/Input.tsx';

<CodeExample code={InputCode}>
  <Input />
</CodeExample>

### Bare

Remove borders and padding using the `isBare` prop.

import Bare from '../../examples/forms/inputs/Bare.tsx';
import BareCode from '!!raw-loader!../../examples/forms/inputs/Bare.tsx';

<CodeExample code={BareCode}>
  <Bare />
</CodeExample>

### Disabled

A disabled Input prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may not read
aloud by a screenreader.

import Disabled from '../../examples/forms/inputs/Disabled.tsx';
import DisabledCode from '!!raw-loader!../../examples/forms/inputs/Disabled.tsx';

<CodeExample code={DisabledCode}>
  <Disabled />
</CodeExample>

### Faux input

Provide input styling without native input backing.

import FauxInput from '../../examples/forms/inputs/FauxInput.tsx';
import FauxInputCode from '!!raw-loader!../../examples/forms/inputs/FauxInput.tsx';

<CodeExample code={FauxInputCode}>
  <FauxInput />
</CodeExample>

### Focus inset

Set focus styling inside the element using the `focusInset` prop. (Useful when an Input is flush
with another component.)

import FocusInset from '../../examples/forms/inputs/FocusInset.tsx';
import FocusInsetCode from '!!raw-loader!../../examples/forms/inputs/FocusInset.tsx';

<CodeExample code={FocusInsetCode}>
  <FocusInset />
</CodeExample>

### Hint text

To provide additional context, use hint text with the label.

import Hint from '../../examples/forms/inputs/Hint.tsx';
import HintCode from '!!raw-loader!../../examples/forms/inputs/Hint.tsx';

<CodeExample code={HintCode}>
  <Hint />
</CodeExample>

### Media input

Media elements add even more context to an input.

import Icons from '../../examples/forms/inputs/MediaInput.tsx';
import IconsCode from '!!raw-loader!../../examples/forms/inputs/MediaInput.tsx';

<CodeExample code={IconsCode}>
  <Icons />
</CodeExample>

### Placeholder

Placeholder text provides the user with example content or additional context.

import Placeholder from '../../examples/forms/inputs/Placeholder.tsx';
import PlaceholderCode from '!!raw-loader!../../examples/forms/inputs/Placeholder.tsx';

<CodeExample code={PlaceholderCode}>
  <Placeholder />
</CodeExample>

### Read-only

A read-only Input prevents user modification but allows selection.

import ReadOnly from '../../examples/forms/inputs/ReadOnly.tsx';
import ReadOnlyCode from '!!raw-loader!../../examples/forms/inputs/ReadOnly.tsx';

<CodeExample code={ReadOnlyCode}>
  <ReadOnly />
</CodeExample>

### Size

An Input can be default or compact.

import Size from '../../examples/forms/inputs/Size.tsx';
import SizeCode from '!!raw-loader!../../examples/forms/inputs/Size.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

### Textarea input

A Textarea is for multiline text. Textarea height is fixed by default but can be
configured to resize automatically based on minimum and maximum rows or resize
manually with the `isResizable` prop.

import Textarea from '../../examples/forms/inputs/Textarea.tsx';
import TextareaCode from '!!raw-loader!../../examples/forms/inputs/Textarea.tsx';

<CodeExample code={TextareaCode}>
  <Textarea />
</CodeExample>

### Validation

Show success, warning, and danger validation messages with the Message component.

import Validation from '../../examples/forms/inputs/Validation.tsx';
import ValidationCode from '!!raw-loader!../../examples/forms/inputs/Validation.tsx';

<CodeExample code={ValidationCode}>

## How to use it

### Bold

`isBold` applies a bold typography style without conveying any extra emphasis or importance.

import Bold from '../../examples/typography/span/Bold.tsx';
import BoldCode from '!!raw-loader!../../examples/typography/span/Bold.tsx';

<CodeExample code={BoldCode}>
  <Bold />
</CodeExample>

### Hue

You can change the color of the span by specifying a different `hue`.

import Hue from '../../examples/typography/span/Hue.tsx';
import HueCode from '!!raw-loader!../../examples/typography/span/Hue.tsx';

<CodeExample code={HueCode}>
  <Hue />
</CodeExample>

### Icons

You can add icons to Spans.

To include an icon at the beginning of a span, use `<Span.StartIcon>`.

import StartIcon from '../../examples/typography/span/StartIcon.tsx';
import StartIconCode from '!!raw-loader!../../examples/typography/span/StartIcon.tsx';

<CodeExample code={StartIconCode}>
  <StartIcon />
</CodeExample>

Use `<Span.Icon>` to add icons in the middle or at the end of a Span.

import Icon from '../../examples/typography/span/Icon.tsx';
import IconCode from '!!raw-loader!../../examples/typography/span/Icon.tsx';

<CodeExample code={IconCode}>
  <Icon />
</CodeExample>

### Monospace

Use `isMonospace` to apply a monospaced type treatment.

import Monospace from '../../examples/typography/span/Monospace.tsx';
import MonospaceCode from '!!raw-loader!../../examples/typography/span/Monospace.tsx';

<CodeExample code={MonospaceCode}>
  <Monospace />
</CodeExample>

## How to use it

### Default

By default, an Alert will be read first by a screen reader and is persistent
until the user dismisses it.

import Default from '../../examples/notifications/alerts/AlertDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/notifications/alerts/AlertDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Type

Types indicate what kind of message is in the alert and help the user understand
how to respond. They can be Success, Warning, Danger, or Info.

import Type from '../../examples/notifications/alerts/AlertType.tsx';
import TypeCode from '!!raw-loader!../../examples/notifications/alerts/AlertType.tsx';

  <Type />
</CodeExample>

## How to use it

### Default

The Anchor component is a styled `<a>` tag. It accepts all standard anchor attributes signals a way
to navigate.

import AnchorDefault from '../../examples/anchor/AnchorDefault.tsx';
import AnchorDefaultCode from '!!raw-loader!../../examples/anchor/AnchorDefault.tsx';

<CodeExample code={AnchorDefaultCode}>
  <AnchorDefault />
</CodeExample>

### External Anchor

An icon appended to the end of an Anchor signals that the link will open in a new window or tab.

import AnchorExternal from '../../examples/anchor/AnchorExternal.tsx';
import AnchorExternalCode from '!!raw-loader!../../examples/anchor/AnchorExternal.tsx';

<CodeExample code={AnchorExternalCode}>
  <AnchorExternal />
</CodeExample>

### Danger

Danger styling communicates that navigating away from your current location will have destructive
results.

import AnchorDanger from '../../examples/anchor/AnchorDanger.tsx';
import AnchorDangerCode from '!!raw-loader!../../examples/anchor/AnchorDanger.tsx';

<CodeExample code={AnchorDangerCode}>
  <AnchorDanger />
</CodeExample>

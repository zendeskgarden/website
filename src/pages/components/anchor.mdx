## Examples

### Default

The Anchor component is a styled `<a>` tag. It accepts all standard anchor attributes and is used as
a link to signal navigation.

import AnchorDefault from '../../examples/anchor/AnchorDefault.tsx';
import AnchorDefaultCode from '!!raw-loader!../../examples/anchor/AnchorDefault.tsx';

<CodeExample code={AnchorDefaultCode}>
  <AnchorDefault />
</CodeExample>

### External Anchor

Inserts an icon at the end of the link to signal when a link opens in a new window or tab.

import AnchorExternal from '../../examples/anchor/AnchorExternal.tsx';
import AnchorExternalCode from '!!raw-loader!../../examples/anchor/AnchorExternal.tsx';

<CodeExample code={AnchorExternalCode}>
  <AnchorExternal />
</CodeExample>

### Danger

Applies danger styling to communicate that navigating away from your location will have destructive
results.

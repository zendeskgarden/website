## How to use it

### Default

A Tooltip Modal must contain a header and a way to exit the experience.

import Default from '../../examples/tooltip-modal/TooltipModalDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/tooltip-modal/TooltipModalDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Placement

A Tooltip modal component can contain a call-to-action, an icon, and
a progress indicator.

There are 13 placement options available. The tooltip modal will occupy
the top position by default. `auto` will default it to a position with
the most available space.

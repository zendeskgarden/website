## How to use it

### Default

The typical usage of a Datepicker component.

import Default from '../../examples/datepicker/DatepickerDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/datepicker/DatepickerDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Custom date format

Dates are formatted using the
[Intl.DateTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DateTimeFormat)
utility. For other formats, consider libraries like
[date-fns](https://date-fns.org/) or [momentjs](https://momentjs.com/).

import Custom from '../../examples/datepicker/DatepickerCustom.tsx';
import CustomCode from '!!raw-loader!../../examples/datepicker/DatepickerCustom.tsx';

<CodeExample code={CustomCode}>
  <Custom />
</CodeExample>

### Datepicker range

Datepicker range works in combination with two text inputs. It displays a
calendar that lets users choose a start and end date.

import Range from '../../examples/datepicker/DatepickerRange.tsx';
import RangeCode from '!!raw-loader!../../examples/datepicker/DatepickerRange.tsx';

<CodeExample code={RangeCode}>
  <Range />
</CodeExample>

### Localization

Dates are localized using the
[Intl.DateTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DateTimeFormat)
utility. For other formats, use `customParseDate` prop, and formatting libraries
like [date-fns](https://date-fns.org/) or [momentjs](https://momentjs.com/).

import Localization from '../../examples/datepicker/DatepickerLocalization.tsx';
import LocalizationCode from '!!raw-loader!../../examples/datepicker/DatepickerLocalization.tsx';

<CodeExample code={LocalizationCode}>
  <Localization />
</CodeExample>

### Size

Datepicker can be default or compact.

Options before and after a certain date can be disabled, limiting users to a specific time window.

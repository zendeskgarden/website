## How to use it

### Default

The typical usage of a Date picker component.

import Default from '../../examples/date-picker/DatepickerDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/date-picker/DatepickerDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Custom date format

Dates are formatted using the
[Intl.DateTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DateTimeFormat)
utility. For other formats, consider using a library like [date-fns](https://date-fns.org/).

import Custom from '../../examples/date-picker/DatepickerCustom.tsx';
import CustomCode from '!!raw-loader!../../examples/date-picker/DatepickerCustom.tsx';

<CodeExample code={CustomCode}>
  <Custom />
</CodeExample>

### Date range

Date range picker works in combination with two text inputs. It displays a
calendar that lets users choose a start and end date.

import Range from '../../examples/date-picker/DatepickerRange.tsx';
import RangeCode from '!!raw-loader!../../examples/date-picker/DatepickerRange.tsx';

<CodeExample code={RangeCode}>
  <Range />
</CodeExample>

### Localization

Dates are localized using the
[Intl.DateTimeFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DateTimeFormat)
utility. For other formats, use `customParseDate` prop, and formatting libraries
like [date-fns](https://date-fns.org/) or [momentjs](https://momentjs.com/).

import Localization from '../../examples/date-picker/DatepickerLocalization.tsx';
import LocalizationCode from '!!raw-loader!../../examples/date-picker/DatepickerLocalization.tsx';

<CodeExample code={LocalizationCode}>
  <Localization />
</CodeExample>

### Size

Date picker can be default or compact.

import Size from '../../examples/date-picker/DatepickerSize.tsx';

Options before and after a certain date can be disabled, limiting users to a specific time window.

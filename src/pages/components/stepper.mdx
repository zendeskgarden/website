## How to use it

### Default

The default usage of a Stepper component.

</CodeExample>

### Layout

A Stepper is vertical by default but can be horizontal using `isHorizontal`.

import Layout from '../../examples/accordions/stepper/StepperLayout.tsx';
import LayoutCode from '!!raw-loader!../../examples/accordions/stepper/StepperLayout.tsx';

<CodeExample code={LayoutCode}>
  <Layout />
</CodeExample>

### Icons

Label numbers can be replaced with icons.

import IconsCode from '!!raw-loader!../../examples/accordions/stepper/StepperIcons.tsx';

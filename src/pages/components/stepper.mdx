## API

The Stepper component follows this structure:

```jsx
<Stepper>
  <Stepper.Step>
    <Stepper.Label />
    <Stepper.Content />
  </Stepper.Step>
  {/* other steps */}
</Stepper>
```

<!-- markdownlint-disable no-emphasis-as-heading -->

**Stepper orientation**

<!-- markdownlint-enable no-emphasis-as-heading -->

- Content for a vertical stepper must be defined using the `Stepper.Content` for each step
- Content for a horizontal stepper must be rendered externally. See [example](#layout) for details.

#### Stepper.Content

<Component components={COMPONENTS} componentName="stepper.content" />

The Content component renders the step's content and provides the calculated styles for animations. Nest it
within a [Step](#stepperstep) component.

#### Stepper.Label

<Component components={COMPONENTS} componentName="stepper.label" />

The Label component renders the provided label text with the step number or the provided icon. It must be
nested within a [Step](#stepperstep).

<PropSheet components={COMPONENTS} componentName="stepper.label" />

#### Stepper.Step

<Component components={COMPONENTS} componentName="stepper.step" />

The Step component must wrap each step's [Label](#stepperlabel) and the corresponding
[Content](#steppercontent) together.

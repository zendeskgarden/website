## How to use it

### Default

A Progress loader represents the percent complete from 0 to 100.

import Default from '../../examples/loaders/progress/ProgressDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/loaders/progress/ProgressDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Color

### Size

A Progress loader can be small, medium, or large in size. It is medium by default.

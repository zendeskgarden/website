## How to use it

### Default

A Progress loader represents the percent complete from 0 to 100.

import Default from '../../examples/loaders/progress/ProgressDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/loaders/progress/ProgressDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Color

The default foreground color of the Progress loader bar is the success color. It
can be changed.

import Color from '../../examples/loaders/progress/ProgressColor.tsx';
import ColorCode from '!!raw-loader!../../examples/loaders/progress/ProgressColor.tsx';

<CodeExample code={ColorCode}>
  <Color />
</CodeExample>

### Size

A Progress loader is medium by default. It can also be small or large.

import Size from '../../examples/loaders/progress/ProgressSize.tsx';
import SizeCode from '!!raw-loader!../../examples/loaders/progress/ProgressSize.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>

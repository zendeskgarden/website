## How to use it

### Block quote

Use Blockquote to represent a body of text that is from another source.

import Blockquote from '../../examples/typography/Blockquote.tsx';
import BlockquoteCode from '!!raw-loader!../../examples/typography/Blockquote.tsx';

<CodeExample code={BlockquoteCode}>
  <Blockquote />
</CodeExample>

### Ellipsis text

Use Ellipsis to automatically provide text content with a native title attribute and text-overflow
styling.

import Ellipsis from '../../examples/typography/Ellipsis.tsx';
import EllipsisCode from '!!raw-loader!../../examples/typography/Ellipsis.tsx';

<CodeExample code={EllipsisCode}>
  <Ellipsis />
</CodeExample>

### Font modifiers

You can apply two modifiers to type: bold and monospace.

import Modifiers from '../../examples/typography/Modifiers.tsx';
import ModifiersCode from '!!raw-loader!../../examples/typography/Modifiers.tsx';

<CodeExample code={ModifiersCode}>
  <Modifiers />
</CodeExample>

### Type scale

Type components come in small, medium, large, extra-large, extra-extra-large, and
extra-extra-extra-large.

import TypeScale from '../../examples/typography/TypeScale.tsx';
import TypeScaleCode from '!!raw-loader!../../examples/typography/TypeScale.tsx';

<CodeExample code={TypeScaleCode}>
  <TypeScale />
</CodeExample>

## How to use it

### Default

Select works with the [Menu](/components/menu) component to display available options.

import SelectDefault from '../../examples/dropdowns/select/Default.tsx';
import SelectDefaultCode from '!!raw-loader!../../examples/dropdowns/select/Default.tsx';

<CodeExample code={SelectDefaultCode}>
  <SelectDefault />
</CodeExample>

### Disabled

A disabled Select prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may
not be read aloud by a screenreader.

import SelectDisabled from '../../examples/dropdowns/select/Disabled.tsx';
import SelectDisabledCode from '!!raw-loader!../../examples/dropdowns/select/Disabled.tsx';

  <SelectDisabled />
</CodeExample>

## API

The Select component follows this structure:

```jsx
<Dropdown>
  <Field>
    <Label />
    <Hint />
    <Select />
    <Message />
  </Field>
  <Menu>
    <Item />
    <Item />
    {/* etc. */}
  </Menu>
</Dropdown>
```

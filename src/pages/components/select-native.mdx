## How to use it

### Default

Select uses a native menu to display available options.

import Default from '../../examples/forms/select/Default.tsx';
import DefaultCode from '!!raw-loader!../../examples/forms/select/Default.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Disabled

A disabled Select prevents user interaction. It doesn’t appear in the tab order, can’t receive focus, and may not read
aloud by a screenreader.

import Disabled from '../../examples/forms/select/Disabled.tsx';
import DisabledCode from '!!raw-loader!../../examples/forms/select/Disabled.tsx';

<CodeExample code={DisabledCode}>
  <Disabled />
</CodeExample>

### Hidden label

## API

A `Draggable` component can be used independently from a `DraggableList`.
They follow this structure:

```jsx
<DraggableList>
  <DraggableList.Item>
    <Draggable>
      <Draggable.Grip />
      <Draggable.Content>{/* content */}</Draggable.Content>
    </Draggable>
  </DraggableList.Item>
  {/* other items */}
</DraggableList>
```

The `Dropzone` component may or may not be a sibling to the `Draggable` component.
It follows this structure:

```jsx
<Dropzone>
  <Dropzone.Icon>
    <Icon />
  </Dropzone.Icon>
  <Dropzone.Message>{/* message */}</Dropzone.Message>
</Dropzone>
```

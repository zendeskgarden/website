## How to use it

### Default

The Color picker allows users to choose colors using sliders and input fields.

import ColorPickerDefault from '../../examples/color-picker/ColorPickerDefault.tsx';
import ColorPickerDefaultCode from '!!raw-loader!../../examples/color-picker/ColorpickerDefault.tsx';

<CodeExample code={ColorPickerDefaultCode}>
  <ColorPickerDefault />
</CodeExample>

### Custom trigger

The Color picker can be displayed using a trigger, e.g. an icon button.

import ColorPickerDialogCustomTrigger from '../../examples/color-picker/ColorpickerDialogCustomTrigger.tsx';
import ColorPickerDialogCustomTriggerCode from '!!raw-loader!../../examples/color-picker/ColorpickerDialogCustomTrigger.tsx';

<CodeExample code={ColorPickerDialogCustomTriggerCode}>
  <ColorPickerDialogCustomTrigger />
</CodeExample>

### Dialog

The Color picker can be shown inside a dialog using `<ColorPickerDialog>`.

import ColorPickerDialogDefault from '../../examples/color-picker/ColorpickerDialogDefault.tsx';
import ColorPickerDialogDefaultCode from '!!raw-loader!../../examples/color-picker/ColorpickerDialogDefault.tsx';

<CodeExample code={ColorPickerDialogDefaultCode}>
  <ColorPickerDialogDefault />
</CodeExample>

### Form input

The Color picker dialog can be used in an input group.

import ColorPickerDialogFormInput from '../../examples/color-picker/ColorpickerDialogFormInput.tsx';
import ColorPickerDialogFormInputCode from '!!raw-loader!../../examples/color-picker/ColorpickerDialogFormInput.tsx';

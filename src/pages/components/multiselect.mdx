## API

The Multiselect component follows this structure:

```jsx
<Dropdown>
  <Field>
    <Label />
    <Hint />
    <Multiselect />
  </Field>
  <Menu>
    <Item />
    <Item />
    {/* etc. */}
  </Menu>
</Dropdown>
```

### Multiselect

<Component components={props.data.mdx.components} componentName="multiselect" />

The Multiselect component applies state and accessibility attributes to its children. Nest it within a
[Field](#field) component. It renders a customizable `tag` for each selected item. This tag can be any element,
but the surrounding field is designed to work with the [Tag](/components/tags) component.

## How to use it

### Default

Spinners communicate a loading state for an unspecified amount of time.

import Default from '../../examples/loaders/spinner/SpinnerDefault.tsx';
import DefaultCode from '!!raw-loader!../../examples/loaders/spinner/SpinnerDefault.tsx';

<CodeExample code={DefaultCode}>
  <Default />
</CodeExample>

### Color

By default, color is inherited from the parent element. Color can be explicitly set.

import Color from '../../examples/loaders/spinner/SpinnerColor.tsx';
import ColorCode from '!!raw-loader!../../examples/loaders/spinner/SpinnerColor.tsx';

<CodeExample code={ColorCode}>
  <Color />
</CodeExample>

### Size

By default, the size is inherited from the font size of the parent element. Size can be explicitly set.

import Size from '../../examples/loaders/spinner/SpinnerSize.tsx';
import SizeCode from '!!raw-loader!../../examples/loaders/spinner/SpinnerSize.tsx';

<CodeExample code={SizeCode}>
  <Size />
</CodeExample>
